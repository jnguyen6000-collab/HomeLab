# Home Lab

## Overview

A personal cybersecurity lab built on a virtualized Ubuntu environment 
using UTM on macOS. This lab serves as a safe, isolated sandbox to 
practice and document real security concepts without risk to my main 
system. All work here is self-directed and built on top of what I've 
learned in my Cybersecurity Administration degree.

## Environment
UTM

 OS: Ubuntu (Linux)

## Projects

| Folder | Description |
|--------|-------------|
| [NetworkScan](./NetworkScan) | Network discovery and enumeration using Nmap. Includes host scanning, port identification, and service detection. |
| [Log Analysis](./Log%20Analysis) | Linux log parsing using command line tools including grep and awk to identify patterns and anomalies. |
| [Firewall_config](./Firewall_config) | Configuring and managing firewall rules using UFW to control inbound and outbound traffic. |

## Skills Practiced
Nmap · grep · awk · UFW · Linux CLI · Network Scanning · 
Log Analysis · Firewall Configuration
